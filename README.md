# Taskbar
## Overview
This is a C# WinForms Class Library that creates a toolbar showing system resource usage.

Resources include:

* CPU Usage/Temperature
* GPU Usage/Temperature
* RAM Usage
* Disk Read/Write Rate
* Network Upload/Download Rate
